## Summary
在深度学习模型训练中，学习率（Learning Rate）的选择与调度至关重要。学习率不仅决定了参数更新的步长，直接影响模型能否收敛或达到性能上限，更是一个需要随训练过程动态调整的超参数。视频详细讲解了从预热（Warm-up）、余弦退火（Cosine Annealing）到One Cycle等多种主流调度策略，探讨了如何根据任务规模、训练成本和数据特性，平衡训练稳定性与收敛速度，从而显著优化模型的最终表现。


### Highlights
- 学习率调度通过节奏性调整步长，能有效避免模型训练初期的不稳定，并提升后期参数优化的精度。
- 采用线性预热（Warm-up）机制，可以让模型在训练初期平稳过渡，防止因梯度过大导致训练过程初期崩溃。
- 余弦退火（Cosine Annealing）是目前大模型训练的标配，其衰减曲线符合“先慢后快再慢”的逻辑，兼顾了快速探索与精细优化。
- 带重启（Restart）的余弦退火策略虽然通过强制跳出局部极小值提升了潜力，但因训练风险较高，在大规模预训练中通常被更稳定的策略所取代。
- One Cycle调度策略将更新频率从按轮次（Epoch）调整为按批次（Batch），在算力预算有限且需快速达到基准线（Baseline）的场景中表现卓越。

### Questions
1. **[为什么在训练神经网络时，学习率不能从头到尾保持不变？](https://bibigpt.co/search?q=%E4%B8%BA%E4%BB%80%E4%B9%88%E5%9C%A8%E8%AE%AD%E7%BB%83%E7%A5%9E%E7%BB%8F%E7%BD%91%E7%BB%9C%E6%97%B6%EF%BC%8C%E5%AD%A6%E4%B9%A0%E7%8E%87%E4%B8%8D%E8%83%BD%E4%BB%8E%E5%A4%B4%E5%88%B0%E5%B0%BE%E4%BF%9D%E6%8C%81%E4%B8%8D%E5%8F%98%EF%BC%9F)**
   - 固定学习率往往导致模型要么在最优解附近震荡（步长过大），要么陷入极其缓慢的收敛过程（步长过小），无法在训练末期进行精细调整。通过调度策略动态减小学习率，可以在保证初期训练速度的同时，实现最终模型精度的最大化。
- 2. **[如果遇到训练过程停滞，该如何选择学习率衰减策略？](https://bibigpt.co/search?q=%E5%A6%82%E6%9E%9C%E9%81%87%E5%88%B0%E8%AE%AD%E7%BB%83%E8%BF%87%E7%A8%8B%E5%81%9C%E6%BB%9E%EF%BC%8C%E8%AF%A5%E5%A6%82%E4%BD%95%E9%80%89%E6%8B%A9%E5%AD%A6%E4%B9%A0%E7%8E%87%E8%A1%B0%E5%87%8F%E7%AD%96%E7%95%A5%EF%BC%9F)**
   - 若训练任务较陌生或缺乏先验经验，可使用根据验证集效果自动触发的“平台期衰减”策略，在模型不再进步时手动调低学习率；若训练预算充足或追求模型表现，推荐优先使用“预热+余弦退火”的组合方案，这是目前业界公认最稳定高效的基准选择。


### Terminology
- **超参数 (Hyperparameters)**: 在训练前手动设定，而非由模型通过数据自动学习的参数，如学习率、批次大小（Batch Size）和层数等。
- **预热 (Warm-up)**: 在训练初期使用极小的学习率，并逐步增加至目标值，旨在让模型参数和动量在训练启动阶段快速进入稳定状态。
- **余弦退火 (Cosine Annealing)**: 一种基于余弦函数进行学习率衰减的策略，其特点是衰减节奏平滑，前期学习率较高以便快速收敛，后期平缓下降利于精细化调整。
- **One Cycle策略**: 一种将学习率调整频率细化到每个Batch的策略，配合动量的反向关联，能显著加速模型在较短训练周期内的收敛。

### [04:22](https://aitodo.co/content/e0380d66-4186-4caf-9e89-0e580efc5043?t=262.000) - 🚀 预热（Warm-up）：稳定起步的关键

![章节截图 04:22](https://bibigpt-apps.chatvid.ai/screenshots/bilibili.com/BV1NMVn6vE7o/262.jpg)

在训练初期，模型参数尚未稳定，直接使用大的学习率容易导致损失函数发散。视频详细介绍了线性预热（Warm-up）技术，通过在训练前若干步骤内将学习率从一个小值平滑提升至预设目标值，使优化器和动量估计先行趋于稳定。这种设计在现代大模型训练中几乎是标配，它能避免训练初期的震荡，确保后续训练过程的平滑收敛，通过对比实验展示了Warm-up对提升训练稳定性的积极作用。

进一步探讨了特殊场景下的策略。介绍了带有重启的余弦退火，通过周期性提高学习率帮助模型跳出局部最小值，但在大模型训练中因稳定性问题较少使用。同时，讲解了基于验证集的自动调节方案，该方法不依赖先验计划，根据实时收敛状态决定是否调整学习率。虽然具备较强的自适应性，但依赖额外的验证开销且具有滞后性，适用于训练收敛特征不明确的中小型任务。

### [19:21](https://aitodo.co/content/e0380d66-4186-4caf-9e89-0e580efc5043?t=1161.000) - ⚡ 极速训练：One Cycle策略

![章节截图 19:21](https://bibigpt-apps.chatvid.ai/screenshots/bilibili.com/BV1NMVn6vE7o/1161.jpg)

介绍了针对高性能训练设计的One Cycle策略，其核心在于将学习率调度的单位从Epoch细化至Batch。在训练过程中学习率先升后降，并配合动量反向关联，能够在极短时间内实现更高的模型精度。虽然该策略高度依赖Batch Size的大小，且通常需要精准的训练总步数，但在CV等领域有着广泛应用。视频总结了如何根据不同任务的规模、资源预算及收敛要求，在上述多种策略中做出最优选择。

# 小小学习率竟然能这么显著影响模型训练结果

## Summary
- 在深度学习模型训练中，学习率（Learning Rate）的选择与调度至关重要。学习率不仅决定了参数更新的步长，直接影响模型能否收敛或达到性能上限，更是一个需要随训练过程动态调整的超参数。视频详细讲解了从预热（Warm-up）、余弦退火（Cosine Annealing）到One Cycle等多种主流调度策略，探讨了如何根据任务规模、训练成本和数据特性，平衡训练稳定性与收敛速度，从而显著优化模型的最终表现。
## Highlights
- 学习率调度通过节奏性调整步长，能有效避免模型训练初期的不稳定，并提升后期参数优化的精度。
- 采用线性预热（Warm-up）机制，可以让模型在训练初期平稳过渡，防止因梯度过大导致训练过程初期崩溃。
- 余弦退火（Cosine Annealing）是目前大模型训练的标配，其衰减曲线符合“先慢后快再慢”的逻辑，兼顾了快速探索与精细优化。
- 带重启（Restart）的余弦退火策略虽然通过强制跳出局部极小值提升了潜力，但因训练风险较高，在大规模预训练中通常被更稳定的策略所取代。
- One Cycle调度策略将更新频率从按轮次（Epoch）调整为按批次（Batch），在算力预算有限且需快速达到基准线（Baseline）的场景中表现卓越。

## Terminology
- **超参数 (Hyperparameters)**: 在训练前手动设定，而非由模型通过数据自动学习的参数，如学习率、批次大小（Batch Size）和层数等。
- **预热 (Warm-up)**: 在训练初期使用极小的学习率，并逐步增加至目标值，旨在让模型参数和动量在训练启动阶段快速进入稳定状态。
- **余弦退火 (Cosine Annealing)**: 一种基于余弦函数进行学习率衰减的策略，其特点是衰减节奏平滑，前期学习率较高以便快速收敛，后期平缓下降利于精细化调整。
- **One Cycle策略**: 一种将学习率调整频率细化到每个Batch的策略，配合动量的反向关联，能显著加速模型在较短训练周期内的收敛。
